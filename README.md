===================
Node Pinball Daemon
===================

A lightweight webserver that runs on a virtual cab, serving all kinds of neat
stuff.

Installation
============

1. Get and install Node.js.
2. Clone the repository to somewhere
3. npm install -d
4. node app.js

======
